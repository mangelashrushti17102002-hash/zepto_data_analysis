# Zepto Retail Analytics Project

## Overview
This project focuses on analyzing a real-world Zepto grocery dataset using PostgreSQL and Power BI. The goal of the project is to perform data cleaning, inventory analysis, pricing analysis, discount analysis, and generate business insights through SQL queries and interactive dashboards.

The project demonstrates practical skills in SQL, data analysis, and business intelligence reporting.

---

## Dataset
The dataset contains product-level information from a Zepto grocery inventory dataset.

### Dataset Includes:
- Product Name
- Category
- MRP (Maximum Retail Price)
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Product Weight
- Stock Availability
