# Zepto Retail SQL Analytics Project

## Project Overview
This is a complete, real-world data analytics project based on an e-commerce inventory dataset scraped from Zepto — one of India’s fastest-growing quick-commerce platforms. The project simulates real analyst workflows, starting from raw data import and cleaning to SQL-based business analysis and interactive dashboard creation.

The objective of this project is to analyze product inventory, pricing, discounts, stock availability, and category performance using PostgreSQL and Power BI.

This project is ideal for:

📊 Data Analyst aspirants building portfolio projects for internships and job interviews  
📚 Beginners learning SQL and Power BI through hands-on practice  
💼 Candidates preparing for retail, e-commerce, and business analytics roles  
📈 Anyone interested in data cleaning, reporting, and dashboard development 

The project demonstrates practical skills in SQL, data analysis, and business intelligence reporting.

---

## Dataset
The dataset contains product-level information from a Zepto grocery inventory dataset.

### Dataset Includes:
- Product Name
- Category
- MRP (Maximum Retail Price)
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Product Weight
- Stock Availability
